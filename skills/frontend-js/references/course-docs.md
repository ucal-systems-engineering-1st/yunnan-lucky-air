## Resumen de requisitos mínimos EE3

Extraído de los lineamientos EE3 — sección 2:

| Requisito | Detalle |
|-----------|---------|
| Script externo | `/scripts/main.js` correctamente enlazado |
| Interacciones DOM | Mínimo 2 funcionalidades que respondan a acciones del usuario |
| Validación JS | Interceptar submit, `preventDefault()`, mensajes de error/éxito |
| localStorage | Mínimo 1 caso: guardar y recuperar preferencia/estado |
| Fetch + render | Datos desde `/data/site-data.json` o endpoint; render dinámico; manejo de errores |
| GitHub | Commits por hito: DOM/eventos, validación, storage, fetch/render + README actualizado |
