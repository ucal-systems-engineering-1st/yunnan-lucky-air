## Resumen de requisitos mínimos EE1

Extraído de los lineamientos EE1 — sección 2:

| Requisito | Detalle |
|-----------|---------|
| Estructura HTML | `head` con meta charset/viewport/title, secciones coherentes |
| Navegación funcional | Menú con anclas internas + enlaces externos sin rotos |
| Imágenes | Con atributo `alt` descriptivo |
| Lista | Mínimo 5 ítems |
| Tabla | Mínimo 3 columnas × 4 filas |
| Formulario | Campos: nombre, correo, motivo/mensaje; `label`+`for/id`; `required`, `type="email"` |
| GitHub | Repositorio con mínimo 4 commits significativos + README |
