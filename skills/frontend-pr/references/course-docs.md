## Rúbrica EE4 — Dominio de plataformas y herramientas (Control de versiones)

| Nivel | Descripción |
|-------|-------------|
| **Sobresaliente (5)** | Gestiona el control de versiones de forma autónoma: ramas, pull requests y resolución básica de conflictos para integrar cambios de manera ordenada y verificable |
| **Notable (4)** | Aplica comandos básicos de control de versiones bajo supervisión, utilizando un repositorio en la nube para almacenar y actualizar su propio código |
| **Aprobado (3)** | Aplica comandos básicos con apoyo puntual, con inconsistencias en la frecuencia o claridad de los commits |
