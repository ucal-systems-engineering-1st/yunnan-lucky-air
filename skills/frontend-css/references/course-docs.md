## Resumen de requisitos mínimos EE2

Extraído de los lineamientos EE2 — sección 2:

| Requisito | Detalle |
|-----------|---------|
| CSS externo | `/css/styles.css` correctamente enlazado con `<link>` |
| Estilos base | Tipografía, colores, espaciados, `line-height` |
| Flexbox | En header/navbar y sección de tarjetas; `justify-content`, `align-items`, `gap`, `flex-wrap` |
| CSS Grid | En galería o sección destacada; `grid-template-columns`, `gap` |
| Responsive | Mínimo 2 breakpoints (móvil + tablet/desktop); sin scroll horizontal |
| GitHub | Commits evidenciando hitos: css-base, Flexbox, Grid, responsive + README actualizado |
