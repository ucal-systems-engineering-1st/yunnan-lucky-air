## **II. SUMILLA**

Las empresas e instituciones requieren profesionales capaces de diseñar, construir y mantener interfaces web para comunicar información, promover servicios y habilitar experiencias digitales en contextos laborales como equipos de producto, marketing digital, áreas de comunicación, emprendimientos y desarrollo de software, donde se demanda crear y actualizar sitios web de manera ágil y colaborativa.

En ese sentido, el curso es importante porque fortalece el perfil de egreso al desarrollar la capacidad de construir soluciones web básicas, gestionar el código y la colaboración mediante control de versiones, verificar y corregir el funcionamiento del producto, y organizar la estructura del proyecto para asegurar su mantenibilidad y evolución.

De esta manera, la naturaleza del curso es práctica y abordará los siguientes ejes temáticos: fundamentos de HTML y estructuración del contenido, estilización y maquetación con CSS (Flexbox, Grid y responsive), programación con JavaScript para interactividad (DOM, eventos, validación, localStorage, consumo de datos con Fetch y render dinámico), y uso de Git y GitHub para control de versiones, publicación y despliegue básico.

## **IV. RESULTADOS DE APRENDIZAJE**

**RA1:** Al finalizar el curso, el estudiante será capaz de desarrollar un sitio web estático funcional, estructurado con HTML semántico y formularios validados, gestionando el código fuente mediante Git y GitHub, aplicando buenas prácticas de versionamiento, documentación y validación técnica para una entrega evaluable y colaborativa.

**RA2:** Al finalizar el curso, el estudiante será capaz de diseñar un sitio web responsive, aplicando hojas de estilo CSS, técnicas de layout con Flexbox y Grid, y reglas de adaptación a distintos dispositivos, asegurando una estructura visual coherente, escalable y alineada a buenas prácticas de desarrollo frontend.

**RA3:** Al finalizar el curso, el estudiante será capaz de implementar funcionalidades interactivas y dinámicas en un sitio web mediante JavaScript, asegurando un comportamiento funcional, verificable y alineado a buenas prácticas de desarrollo frontend.

**RA4:** Al finalizar el curso, el estudiante será capaz de integrar un proyecto front-end completo, organizando el código de forma modular, gestionando el trabajo colaborativo con control de versiones, aplicando criterios de calidad, accesibilidad y verificación funcional, y publicando el producto final con documentación técnica para su uso y evaluación.

## **VI. METODOLOGÍA**

El curso se desarrolla bajo un enfoque práctico y orientado al aprendizaje basado en proyectos. La metodología promueve la aplicación inmediata de los contenidos mediante la construcción progresiva de una página web funcional, que evoluciona en complejidad durante el semestre.

Las sesiones se desarrollan de manera síncrona, combinando explicación conceptual breve, demostraciones guiadas (live coding), resolución de ejercicios aplicados y desarrollo supervisado del proyecto. El docente cumple un rol de facilitador técnico, brindando acompañamiento y retroalimentación constante para fortalecer la correcta implementación del código y la comprensión de los fundamentos del desarrollo frontend.

El curso avanza de forma secuencial abordando la estructuración con HTML, estilización y maquetación con CSS, programación con JavaScript para la interactividad y el uso de Git y GitHub para el control de versiones y publicación básica.

La evaluación se organiza en cuatro Entregables de Evidencia (EE1, EE2, EE3 y EE4), que se desarrollan de manera progresiva y articulada, conformando un proyecto final integrador.

Al desarrollarse bajo la modalidad virtual, las sesiones se gestionan a través del aula virtual, donde los estudiantes podrán consultar el sílabo, cronograma, lineamientos y rúbricas de evaluación, garantizando claridad en el proceso formativo y en los criterios de evaluación.
